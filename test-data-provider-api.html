故障排除指南

常见错误： Cannot read properties of undefined (reading 'sendMessage')

这意味着脚本无法访问chrome.runtime.sendMessage方法，通常是环境问题导致的。

可能的原因和解决方案：

  1. 使用正确的浏览器
    确保使用Chrome或基于Chromium的浏览器（如Edge）。
  2. 不要直接打开本地文件
    Chrome的安全限制不允许本地HTML文件通过file://协议直接与扩展通信。请通过以下方式之一访问测试页面：
    • 使用Web服务器（如http-server、Python的SimpleHTTPServer等）
    • 将这些文件上传到已部署的网站
    • 或者直接将测试代码粘贴到Chrome扩展的DevTools控制台中运行
  3. 检查扩展是否安装并启用
    确认Page Assist扩展已正确安装并启用。可以在Chrome的扩展页面（chrome://extensions/）中查看。
  4. 验证扩展ID
    确认脚本中使用的扩展ID是否正确。当前使用的ID是：ffiigpcapfgmbpcggdfklpikceehbffd
  5. 检查权限配置
    确保Page Assist扩展允许外部连接，特别是对当前域名的连接权限。这通常在扩展的manifest.json中的externally_connectable字段中配置。
  6. 使用扩展内部测试
    如果以上方法都不奏效，请使用以下方式在扩展内部进行测试：
    1. 打开Chrome扩展页面 (chrome://extensions/)
    2. 找到Page Assist扩展，点击"详情"
    3. 找到"查看扩展选项"并点击
    4. 在扩展的选项页面中打开开发者工具 (按F12)
    5. 切换到控制台，粘贴并运行测试脚本
